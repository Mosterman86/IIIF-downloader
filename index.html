IIIF Digital Catalogue — Download Tool

About this tool

This tool provides password-protected access to original image files from the Wolfsonian Digital Catalogue. It works with any IIIF-compliant manifest URL from the collection. Images are rendered via OpenSeadragon 4.1 using the IIIF Presentation API (v2 & v3).

How to find a manifest URL

Open any object record in the Wolfsonian Digital Catalogue. In the image viewer, click the menu icon in the top left to open the sidebar. Then click the “i” icon and scroll to the bottom of that panel, where you will find a Links section containing the IIIF Manifest URL. Copy and paste that full link into the field below, then press Load or hit Enter.

How to download

Once the image loads, a ↓ Download button will appear beneath the viewer. Click it, enter the access password when prompted, then click the download link that appears.

Chrome & Firefox: the file will save directly to your downloads folder.

Safari: the image will open in a new tab. To save it, right-click (or Control-click) anywhere on the image and choose Save Image As… — or use File → Save As from the menu bar.

Usage & copyright
Downloaded files are for staff and institutional use only. All objects remain subject to the Wolfsonian's use and reproduction policies. Before distributing any image, consult the object record for rights and credit line information. By downloading, you accept the Terms of Use.
